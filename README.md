# Мой учебный проект

## Description
Этот проект содержит простое консольное приложение на языке C.
Программа выводит информацию о контрольной работе, учебной группе и студенте.
Она предназначена для демонстрации базовой структуры учебного репозитория.

![Логотип Git](https://git-scm.com/images/logo@2x.png)

- Автоматизировать вывод информации о контрольной работе.
- Упростить сборку и запуск консольного приложения.
- Подготовить базовую документацию по использованию проекта.

> Этот учебный проект показывает базовый процесс разработки и документирования приложения на C.

## Installation
1. Клонируйте репозиторий на локальный компьютер.
2. Перейдите в каталог проекта.
3. Убедитесь, что в системе установлен компилятор GCC.

| Версия | Поддержка |
| --- | --- |
| 1.0.x | Полная |
| 0.9.x | Ограниченная |
| 0.8.x | Нет |

## Usage
Приложение выводит **важные сведения** о контрольной работе и _авторе проекта_.

```bash
gcc src/main.c -o program
./program
```

| Команда | Описание |
| --- | --- |
| `git init` | Инициализация репозитория |
| `git add` | Добавление файлов в индекс |
| `git commit` | Создание коммита |
